# How to Print — Mariscos Los Corchos Signage Pack

## Option 1: Print Individual Signs
1. Navigate to the `signs/` folder
2. Open the desired `.html` file in Chrome or Edge
3. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
4. Settings:
   - Paper size: **Letter (8.5 × 11 in)**
   - Margins: **None** or **Minimum**
   - Scale: **100%**
   - Background graphics: **ON** (required for colors/branding)
5. Click Print

## Option 2: Print the Full Pack
1. Open `full-pack/los-corchos-full-signage-pack.html` in Chrome
2. Click the **🖨 Print All** button in the top nav bar
3. All pages will print in sequence

## Recommended Paper
- **Signs:** Heavy matte paper (32lb+) or cardstock for durability
- **Templates:** Standard 20lb copy paper (re-printed daily/weekly)
- **Laminate** all permanent kitchen signs for longevity

## Recommended Printers
- Any color laser printer for best results
- Home inkjet printers work — use "Best Quality" setting
